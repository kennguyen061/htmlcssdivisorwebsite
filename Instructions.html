For this project you will demonstrate how to accept two numbers, divide the first by the second, and show the result. You will use HTML and JavaScript.

Please complete this project and submit your finished project back to Hayden when you are finished. We will discuss the work you did on this project during the interview.

User Interface (UI) requirements:
  1. Add a textbox to enter a number
  2. Add a another textbox to enter a second number
  3. Add a button labeled "Calculate".
  4. Add a label to show the result.

Business requirements:

When the button is clicked,

  1. When the button is clicked, divide the first number by the second number and show the results in the label.
  2. Use a JavaScript function to perforrm your calculation.
  3. Best practices dictate validating input prior to doing anything with it.
  4. Remember and apply the math rules you know for division and decimal points.
  5. You have completed the Minimum Viable Product (MVP)! You may submit your project now or improve your project with the considerations below.

Considerations:
  1. Labeling inputs helps guide your user to doing the right thing.
  2. What should your program do if it is run multiple times?

Extra Considerations:
  1. Judicious use of color and styling helps highlight what is important on the page.
  2. Having styling live in its own file helps decouple your HTML from the styling.
  3. Having JavaScript live in its own file makes writing automated tests easier.

Bonus Considertions:
  1. Write automated tests for any JavaScript you write. You should provide instructions in .js file how to setup and run the tests.
  2. Flexbox or CSS Grid can make your page responsive (i.e., able to look good on any screen).
